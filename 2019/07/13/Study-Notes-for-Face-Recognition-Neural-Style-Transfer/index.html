Study Notes for 'Face Recognition & Neural Style Transfer'

Week 4 in Andrew Ng’s Convolutional Neural Networks on Coursera.


Face Recognition

What is face recognition?

liveliness detection

Face verification

  • Input image, name/ID
  • Output whether the input image is that of the claimed person

Face recognition

  • Has a database of K persons
  • Get an input image
  • Output ID if the image is any of the K persons(or “not recognized”)

One Shot Learning

Learn from one example to recognize the person again.

To solve it,

Learn a “similarity” function

d(img1, img2) = degree of difference between the 2 images

if d(img1, img2) <= τ (a certain threshold)

​ output “same”

if d(img1, img2) > τ

​ output “different”


Siamese Network

Input an image x(1) into a ConvNet, have a FC output layer that has 128 units and view this as “encoding of x(1)”, which is an image.

Input an image x(2) and get the output.

Compare the two outputs

d(x(1), x(2)) = eucledian distance between x(1) and x(2)

Parameters of NN define f(x(i)) / encoding of x(i)

Learn parameters so that

if x(i) and x(j) are the same person, d(x(i), x(j)) is small

if x(i) and x(j) are different persons, d(x(i), x(j)) is large


Triplet Loss

Learning objective

  • Want the distance between the anchor and positive image be small

  • Want the distance between the anchor and negative image be large

Looking at 3 images at a time

Anchor as ‘A’, Positive as ‘P’, Negative as ‘N’

Want d(A, P) <= d(A, N)

which is equivalent to the equation: d(A, P) - d(A, N) <= 0

but in pratice, since we don’t want it to output a trivial solution(that both encodings are close to or equal to 0), we need to add an extra parameter α, to make sure that the difference between d(A, P) and d(A, N) is sufficiently big. α here is also called margin parameter.

d(A, P) - d(A, N) + α <= 0

Loss function:

Given 3 images A, P, N:

L(A, P, N) = max(d(A, P) - d(A, N) + α, 0)

So long as d(A, P) - d(A, N) + α <=0, the loss is 0.

otherwise, is the equation “d(A, P) - d(A, N) + α <= 0” isn’t satisfied

Loss = d(A, P) - d(A, N) + α, which is a positive loss.

For the training set, we do need multiple images of the same person.

After training this network, the “one shot learning” problem can be solved.

Choosing triplets A, P, N

Choose triplets that’re hard to train on,

that d(A, P) + α <= d(A, N) is hard to be satisfied.


Face Verification and Binary Classification

Alternatively, we can also input a pair of images and output a logistic regression unit(1 or 0) to evaluate if these are of the same person or not.

Precompute the encoding of the images in the database can save lots of computational power.

Use pairs to train the network.


Neural Style Transfer

What is neural style transfer?

Content(C), Style(S), Generated(G)


What are deep ConvNets learning?

Pick a unit in layer 1. Fine the nine image patches that maximize the unit’s activation.

Repeat for other units.

Through this, it can give you a sense of which unit captures what kind of features in the image.

In deeper layers, each unit sees larger image patches.

From shallower to deeper layers, the image patches each unit detects gradually get more complex(from edges to specific objects)


Cost Function

Neural Style Transfer Cost Function

J(G) =α * Jcontent(C, G) + β * Jstyle(S, G)

  • Initialize G randomly

    ​ G = 100 x 100 x3

  • Use gradient descent to minimize J(G)

    It’s updating the pixel values of G.


Content Cost Function

J(G) =α * Jcontent(C, G) + β * Jstyle(S, G)

  • Use hidden layer l to compute content cost (l is usually neither too shallow nor too deep)
  • Use pre-trained ConvNet (E.g., VGG network)
  • Let a[l](C) and a[l](G) be the activations of layer l on the images.
  • If a[l](C) and a[l](G) are similar, both images have similar content

J[l]content(C, G) = (Forbenius norm of (a[l](C) - a[l](G)) ) ^2


Style Cost Function

Meaning of the style of an image:

Say we’re using hidden layer l’s activation to measure “style”.

Define style as correlation between activations across channels.

How often they(the high level features) occur or don’t occur together

Style matrix:

Let a[l]i, j, k = activation at (i, j, k). The dimension of G[l] is nc[l] x nc[l]

k and k’ range from 1 to nc[l]

G[l](S)kk’ = (sum over i, j) * (a[l](S)i, j, k) * (a[l](S)i, j, k’) // style matrix of image S

G[l](G)kk’ = (sum over i, j) * (a[l](G)i, j, k) * (a[l](G)i, j, k’) // style matrix of image G

In linear algebra, this is also called “gram matrix”

For each layer,

J[l]style(S, G) = (Forbenius norm of (G[l](S) - G[l](G)) ) ^2

The overall equation is,

Jstyle(S, G) = (sum over the l layers) (λ[l] * J[l]style(S, G))

Allows to take both low level and high level features into account when computing the style.


1D and 3D Generalizations

1D:

EKG of someone’s heartbeat

e.g. a 14 dimensional thing convolve with a 5 dimensional filter gives a 10 dimensional output

In most cases for 1D data we may use a Recurrent Neural Network though.


3D:

e.g. 14 x 14 x 14 x 1(nc[l]) convolve with a 5 x 5 x 5 x 1(nc[l+1]) filter gives a 10 x 10 x 10 volume.
